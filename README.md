Cucci Feeder
=========

Semplice app Android per visualizzare gli ultimi feed RSS.

  - Cards Layout
  - Visualizza titoli, date e, se presenti nella descrizione, immagini
  - Cliccando sul link viene aperto l'articolo nel Browser
